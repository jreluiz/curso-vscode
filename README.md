# 💻 Curso de VS Code para Iniciantes

> Um curso introdutório e prático sobre o **Visual Studio Code** — do zero ao uso produtivo no dia a dia.

Bem-vindo(a)! Este repositório contém um curso completo sobre o **Visual Studio Code**, o editor de código usado por milhões de desenvolvedores no mundo todo. O material foi pensado para quem está começando: nenhum conhecimento prévio do editor é necessário.

---

## 🎯 Objetivos do curso

Ao final deste material, você será capaz de:

- ✅ Instalar e configurar o VS Code no seu computador (ou usá-lo no laboratório)
- ✅ Navegar com confiança pela interface do editor
- ✅ Editar código com produtividade usando atalhos e multicursores
- ✅ Instalar e gerenciar extensões essenciais
- ✅ Usar o terminal integrado para rodar seus programas
- ✅ Trabalhar com Git e GitHub sem sair do editor
- ✅ Personalizar o editor para o seu fluxo de trabalho

## 🗺️ Como estudar este material

1. **Siga os módulos em ordem** — cada um assume o conhecimento do anterior.
2. **Pratique enquanto lê** — abra o VS Code ao lado e teste cada comando.
3. **Faça os exercícios** — a fixação vem da prática, não da leitura.
4. **Use o cheatsheet** — imprima ou deixe aberto em uma aba durante as aulas.

> 💡 **Dica:** não tente decorar todos os atalhos de uma vez. Escolha 2 ou 3 por semana e use-os até virarem hábito.
